# Intuned Browser SDK

Prebuilt helpers for browser automation — navigation, waiting, extraction, pagination, and file/attachment handling — that make automations easier and more reliable.

## Helper reference (per language)

Each resource lists every available helper function with usage examples and common patterns. Read the one for your project's language:

- **Python** (`intuned_browser`): `resources/python.md`
- **TypeScript** (`@intuned/browser`): `resources/typescript.md`

## When you need more than the reference

For details not in the language reference — full arguments, return types, extra examples, or when a method errors — the official docs are the source of truth. Do NOT read from `venv/`, `node_modules/`, or source files to discover function signatures.

1. `search_intuned` — find the page for the helper you need (e.g. `saveFileToS3`, `wait_for_captcha_solve`).
2. `query_docs_filesystem_intuned` — read that page for full arguments, return types, and examples.
